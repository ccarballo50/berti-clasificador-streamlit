
# Clasificador BERTI - Dolor torácico clínico

Aplicación Streamlit que carga un modelo NLP clínico desde HuggingFace y permite clasificar anamnesis como TIPICO o ATIPICO.

## Repositorio modelo HuggingFace
https://huggingface.co/ccarballo50/berti-clasificador-clinico

## Archivos necesarios
- app_streamlit_bert.py
- requirements.txt

## Instrucciones de despliegue
1. Sube estos archivos a un nuevo repositorio GitHub (público).
2. Ve a https://streamlit.io/cloud y despliega tu app.
3. Parámetros:
   - Repositorio: `usuario/repo`
   - Rama: `main`
   - Archivo principal: `app_streamlit_bert.py`
